# KAVEEP-SIA

KAVEEP System Intelligence Agent (SIA)

A local-first AI Agent designed to understand, analyze, optimize, and safely manage a personal computer.

---

## Mission

> Understand the computer before changing the computer.

---

## Safety Rules

- Do not create destructive actions.
- Do not delete, move, rename, overwrite, or modify real user files.
- Simulation only unless explicitly approved by the user.
