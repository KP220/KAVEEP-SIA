# Project Constitution

Project:

KAVEEP-SIA

Version:

0.1

Mission

Understand the computer before changing the computer.

---

Core Principles

1.
Safety First

No destructive actions.

2.
Human Approval Required

Every real modification requires explicit approval.

3.
Specification First

Never implement before specification.

4.
Evidence Driven

Every decision must be supported by evidence.

5.
Reproducibility

Every experiment must be repeatable.

6.
Transparency

Every action must be logged.

7.
Rollback

Every modification must be reversible.
